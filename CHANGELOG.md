# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [2.2.0] - 2025-02-19

### Added
- Initial public repository for SmartDocs Insight.
- Tkinter-based GUI for invoice analysis and CoO request preparation.
- CSV/PDF pairing with invoice signature and BPID validation.
- COO/weight extraction with EU vs non-EU classification.
- Detection and modal display of missing COO lines.
- Excel tracking sheet export using openpyxl.
- Dark/Light mode with persisted user preferences.
- Basic logging to `SmartDocs.log`.

## [2.1.x] - Internal

- Internal iterations on CSV parsing and GUI layout.
- Improvements to missing COO detection and error handling.
